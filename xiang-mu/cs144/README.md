# CS144

### Reference

> 观点：关于CS144项目，网上有很多lab的讲解和代码；但是其实看很多的讲解意义不大，因为lab本身就没有标准答案，而且看过多的讲解还会使自己陷入一种”寻找最好资料或方法”的困境，难以专注于lab本身。所以项目参考在精不在多，选择一个主要参考即可；其他的参考都是配合主要参考进行更好的理解和优化；
>
>
>
> 结论：选择一个主要参考，其他的参考都是基于主要参考的扩展和优化。
